# uniproc
Universal Procedure Language

## Usage Goals

Rather than compiling to byte code, uniproc will expand into reasonably readable macros for the target language

## Roadmap

* Readline/Input

## Example

```
PRINT Welcome!
ASSIGN @name INPUT What's your name?
PRINT Hello, @name
```
